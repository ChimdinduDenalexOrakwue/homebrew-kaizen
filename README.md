# homebrew-kaizen

Homebrew tap for [kaizen](https://github.com/ChimdinduDenalexOrakwue/kaizen),
the observation-graph CLI for Android agent testing.

```sh
brew tap ChimdinduDenalexOrakwue/kaizen
brew install kaizen
```

Formulae install prebuilt binaries from the kaizen GitHub releases,
verified against the release SHA256SUMS.
